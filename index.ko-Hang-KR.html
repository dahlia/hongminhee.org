오픈 소스

  • Fedify는 TypeScript로 만든 ActivityPub 구현 및 연합우주(fediverse) 서버 프레임워크입니다.
  • Hollo는 ActivityPub을 지원하는 일인 사용자용 경량 마이크로블로그 소프트웨어입니다.
  • LogTape는 Deno/Node.js/Bun/브라우저에서 두루 돌아가는 심플한 로깅 라이브러리입니다.
  • Seonbi는 여러 문장 부호 교정하고 국한문 혼용문을 한글전용문으로 변환해주는 HTML 전처리기입니다.
  • 기타…
    • Libplanet은 탈중앙 멀티플레이어 온라인 게임을 만들기 위한 .NET 라이브러리로, 모든 게임 플레이가 특권적 중앙 서버가 아닌 동등한 노드 사이의 P2P 네트워크에서 이뤄지게 합니다.
    • Bencodex는 BitTorrent의 Bencoding을 확장한 직렬화 포맷입니다. 정규화의 강제(즉, 값과 부호 사이의 대칭성)와 JSON 자료 모델과 비슷한 자료형의 지원을 함께 갖추되, 여전히 원래의 Bencoding과 상위호환됩니다. (즉, 모든 유효한 Bencoding 표현은 유효한 Bencodex 표현이기도 합니다.) 제가 명세를 썼고, Python, TypeScript, .NET, Haskell, PHP와 같은 여러 언어로 구현도 했습니다.
    • libsass-python은 Python용 SASS/SCSS 패키지입니다.
    • Wand는 ctypes 기반의 심플한 Python용 ImageMagick API 바인딩입니다.
    • sphinxcontrib-httpdomain는 HTTP API를 기술하기 위한 Sphinx 도메인입니다.
    • Geofront는 간단한 SSH 키 관리 서비스입니다.
    • Nirum은 마이크로서비스를 위한 IDL 컴파일러이자 RPC 및 분산 객체 프레임워크입니다.
    • Dojang은 크로스플랫폼 닷파일(dotfiles) 관리자입니다.
    • 직지(Jikji)는 정적 사이트 생성기를 만들기 위한 Deno 라이브러리입니다.
    • Wikidata는 Python용 Wikidata 클라이언트 라이브러리입니다.
    • aitertools는 TypeScript에서 비동기 반복자를 다루기 위한 잘 테스트된 유틸리티 함수 모음입니다.
    • json-hash는 JSON 자료를 암호학적으로 다루기 위한 Deno 패키지입니다. JCS, JSON 다이제스트, JSON 머클 트리를 구현합니다.
    • SQLAlchemy-ImageAttach는 엔티티 객체에 이미지를 첨부할 수 있게 해주는 SQLAlchemy 확장입니다.
    • checkmate는 주어진 패치(나 풀 리퀘스트)에서 사람이 한눈에 볼 수 있는 체크리스트를 만들어 주는 작은 프로그램입니다.
    • iterfzf은 명령행 퍼지 파인더인 fzf의 Python 인터페이스입니다.
    • sphinxcontrib-autoprogram은 명령행 프로그램을 자동으로 문서화하는 Sphinx 확장입니다.
    • submark는 주어진 CommonMark/Markdown 문서로부터 특정 부분만 추출하는 작은 명령행 프로그램입니다. (GitHub 액션으로도 사용 가능.)
    • mediawiki-rc-mastodon-bot은 MediaWiki 사이트의 최근바뀜 내용을 Mastodon에 중계하는 작은 프로그램입니다.

제 오픈 소스 관련 활동은 제 GitHub, GitLab, npm, JSR, Hackage, PyPI, NuGet에서도 살펴볼 수 있습니다.
