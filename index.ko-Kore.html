洪民憙

安寧하세요。여기는 서울에서 소프트웨어 프로그래머로 일하는 洪民憙의 웹사이트입니다。저는 自由·오픈 소스 소프트웨어를 만들며、主로 Haskell、Python、TypeScript 같은 言語로 코딩합니다。自由·오픈 소스 소프트웨어와 聯合宇宙의 熱烈한 支持者이기도 합니다。제 職業的 關心事로는 脫中央 P2P 네트워크와 靜的 타입 函數型 프로그래밍 言語 等이 있습니다。(履歷書 →)

오픈 소스

  • Fedify는 TypeScript로 만든 ActivityPub 具顯 및 聯合宇宙(fediverse) 서버 프레임워크입니다。
  • LogTape는 Deno、Node.js、Bun、에지 函數、브라우저에서 두루 돌아가는 심플한 로깅 라이브러리입니다。
  • Hollo는 ActivityPub을 支援하는 一人 使用者用 輕量 마이크로블로그 소프트웨어입니다。
  • BotKit은 ActivityPub 봇을 만들기 爲한 TypeScript 라이브러리입니다。
  • Upyo는 Deno、Node.js、Bun、에지 函數에서 두루 돌아가는 現代的인 이메일 라이브러리입니다。
  • Seonbi는 여러 文章符號 校正하고 國漢文混用文을 한글專用文으로 變換해주는 HTML 前處理機입니다。
  • 其他…
    • Libplanet은 脫中央 멀티플레이어 온라인 게임을 만들기 爲한 .NET 라이브러리로、모든 게임 플레이가 特權的 中央 서버가 아닌 同等한 노드 사이의 P2P 네트워크에서 이뤄지게 합니다。
    • Bencodex는 BitTorrent의 Bencoding을 擴張한 直列化 포맷입니다。正規化의 強制(卽、값과 符號 사이의 對稱性)와 JSON 資料 모델과 비슷한 資料型의 支援을 함께 갖추되、如前히 元來의 Bencoding과 上位互換됩니다。(卽、모든 有效한 Bencoding 表現은 有效한 Bencodex 表現이기도 합니다)제가 明細를 썼고、Python、TypeScript、.NET、Haskell、PHP와 같은 여러 言語로 具顯도 했습니다。
    • libsass-python은 Python用 SASS/SCSS 패키지입니다。
    • Wand는 ctypes 基盤의 심플한 Python用 ImageMagick API 바인딩입니다。
    • sphinxcontrib-httpdomain는 HTTP API를 記述하기 爲한 Sphinx 도메인입니다。
    • Geofront는 簡單한 SSH 키 管理 서비스입니다。
    • Nirum은 마이크로서비스를 爲한 IDL 컴파일러이자 RPC 및 分散 客體 프레임워크입니다。
    • Dojang은 크로스플랫폼 닷파일(dotfiles)管理者입니다。
    • 直指(Jikji)는 靜的 사이트 生成機를 만들기 爲한 Deno 라이브러리입니다。
    • Wikidata는 Python用 Wikidata 클라이언트 라이브러리입니다。
    • aitertools는 TypeScript에서 非同期 反復子를 다루기 爲한 잘 테스트된 유틸리티 函數 모음입니다。
    • json-hash는 JSON 資料를 暗號學的으로 다루기 爲한 Deno 패키지입니다。 JCS、JSON 다이제스트、JSON 머클 트리를 具顯합니다。
    • SQLAlchemy-ImageAttach는 엔티티 客體에 이미지를 添附할 수 있게 해주는 SQLAlchemy 擴張입니다。
    • checkmate는 주어진 패치（나 풀 리퀘스트）에서 사람이 한눈에 볼 수 있는 체크리스트를 만들어 주는 작은 프로그램입니다。
    • iterfzf은 命令行 퍼지 파인더인 fzf의 Python 인터페이스입니다。
    • sphinxcontrib-autoprogram은 命令行 프로그램을 自動으로 文書化하는 Sphinx 擴張입니다。
    • submark는 주어진 CommonMark/Markdown 文書로부터 特定 部分만 抽出하는 작은 命令行 프로그램입니다。(GitHub 액션으로도 使用 可能)
    • mediawiki-rc-mastodon-bot은 MediaWiki 사이트의 最近바뀜 內容을 Mastodon에 中繼하는 작은 프로그램입니다。

제 오픈 소스 關聯 活動은 제 GitHub、GitLab、npm、JSR、Hackage、PyPI、NuGet에서도 살펴볼 수 있습니다。
